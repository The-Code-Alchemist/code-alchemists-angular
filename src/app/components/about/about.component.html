Combining the Mindset of Two Disciplines: Clean Code and Alchemy

✓ The Code part focuses on maintainable clean code. Clean Code is a book written by Robert C. Martin (Uncle Bob) that defines this topic extensively.

An overview of Uncle Bob's books can be found here.

The Software Improvement Group describes 10 guidelines for building maintainable software.

The book is available for free: Building Maintainable Software

✓ The Alchemists part refers to alchemy (al-kimia in Arabic), an ancient proto-science that paved the way for modern science and chemistry in particular.

The goal of Western alchemists was to transform base metals (such as iron, nickel, copper, and lead) into precious metals like silver or gold.

In contrast, curing diseases was the most important goal of Eastern (oriental) alchemists.
